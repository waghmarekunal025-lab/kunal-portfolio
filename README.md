# 🌐 Premium AI Engineer & Developer Portfolio Website

A modern, responsive single-page web portfolio engineered to showcase applications, machine learning systems, and computational linguistics projects. The site is built with a lightweight, zero-dependency frontend layout framework featuring responsive matrix grids, active viewport controls, and interactive client data operations.

🔗 **Live Production URL:** [https://waghmarekunal025-lab.github.io/kunal-portfolio/](https://waghmarekunal025-lab.github.io/kunal-portfolio/)

---

## 🚀 Key Engineering Features & Implementations

* **Dual-Engine Theme Shifter:** Integrates a dark/light visual theme toggle that dynamically modifies global layout tracking variables. Selections are cached inside browser memory via the Web Storage API (`localStorage`) to preserve visual continuity across page reloads.
* **Full-Screen Minimalist Mobile Navigation:** Implements a sliding vertical overlay drawer navigation tray optimized for small smartphone viewports. When activated by the menu toggle loop, it locks core document scroll variables (`overflow: hidden`) to maintain absolute interface containment.
* **Dynamic Typing Staging Layer:** Implements clean textual state shifting across core domains utilizing the `Typed.js` script framework engine to animate specializations interactively.
* **Pointer-Responsive Parallax Cards:** Leverages geometric matrix tracking properties via `Vanilla-Tilt.js` to create dynamic hover micro-interactions across project items without degrading page load tracking metrics.
* **Asynchronous AJAX Contact Pipeline:** Configured with a background `fetch()` request system running behind a modular script event list block. This completely bypasses default cloud page landing redirects, triggering custom checkmark indicators and asynchronous text resets.
* **Instant Clipboard Copy Engine:** Includes embedded JavaScript functions that automatically extract target text values (Email and Phone), copy them directly to the user's local operating system clipboard, and switch icons to display confirmation alerts instantly.

* **Structural Architecture:** Semantic HTML5 Document Staging Elements
